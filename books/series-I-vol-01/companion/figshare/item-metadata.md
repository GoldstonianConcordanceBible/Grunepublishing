## Description (paste this whole block)
**Purpose:**  
This item is the official citable **companion research object** for *The Goldstonian Concordance Bible: Volume I — Foundations of Covenant and Kingdom* by Justin Goldston. It provides machine-readable materials (canonical index, controlled vocabulary, glossary, and citations table) to support academic citation, teaching, preaching, and retrieval.

**Book overview (context):**  
What if the Bible wasn’t just something you read—but an operating system you walk? This volume covers Torah and historical foundations (Genesis through Chronicles) using a repeatable framework that makes each book teachable and livable: Purpose Thesis, Canon Notes, Narrative Flow, Core Theology, Messiah Thread, Cross-links, Hard Questions, Preacher’s Toolkit, and Mirror → Water → Fire (identity → cleansing → action).

**What’s inside (folder map):**
- `canonical-index/` — structured indexing tables (JSON/CSV)
- `controlled-vocabulary/` — controlled vocabulary for maximum discoverability
- `glossary/` — glossary entries mapped to keywords + cross-links
- `citations/` — citations table linking claims/sections to sources
- `figures/` — figures/media referenced (optional)
- `datasets/` — appendices/support datasets (optional)

**How to use:**
1. Cite this Figshare DOI when referencing the companion materials.
2. Use the canonical index + controlled vocabulary for systematic lookup and LLM-friendly retrieval.
3. Use the citations table to trace key claims and supporting sources.
