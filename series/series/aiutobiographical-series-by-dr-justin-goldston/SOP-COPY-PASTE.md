# SOP — Copy/Paste Template Setup (Series Publishing Hub)

## What this is
A repeatable template for setting up a series folder with:
- series-level files (overview, disclaimer, privacy)
- a books folder with 5 book subfolders
- an optional Divine AI Agent pack per book

## Required series-level files
- README.md
- DISCLAIMER.md
- PRIVACY-AND-NAMES.md
- SERIES-OVERVIEW.md
- SOP-COPY-PASTE.md

## Required per-book files (inside each book folder)
- README.md
- amazon-description.md
- keywords.md

## Optional Divine Agent pack (per book)
- README.md
- agent-card.md
- agent-prompt.md
- healing-path.md
- journal.md
- prayer.md
- boundaries.md
- scripture-map.md
- liturgies.md

## Naming rules
- Use lowercase + hyphens for folder names
- Keep book order stable with 01–05 prefixes

## Usage
1) Duplicate the structure
2) Update series overview
3) Update each book’s: README + amazon-description + keywords
4) If using agents: update each agent pack placeholders
5) Commit
