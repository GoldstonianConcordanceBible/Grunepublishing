# Naming Rules

- Book IDs: `series-{ROMAN}-vol-{NN}` (example: `series-I-vol-01`)
- Folder path: `books/{book_id}/`
- Companion pack versioning: Semantic Versioning `MAJOR.MINOR.PATCH`
  - Major = breaking schema change
  - Minor = new content
  - Patch = fixes/typos/metadata corrections
- Use consistent title casing across all files
