# Companion Pack Checklist (Before Figshare Upload)

- [ ] `book-metadata.json` complete (ASIN + date + links)
- [ ] Companion `VERSION.md` set
- [ ] `CITATION.cff` filled (ORCID if available)
- [ ] Controlled vocabulary present (`data/controlled-vocabulary/`)
- [ ] Canonical index present (`data/canonical-index/`)
- [ ] Glossary present (`data/glossary/`)
- [ ] Citations table present (`data/citations/`)
- [ ] Zip/export created (optional)
- [ ] Figshare item created as `dataset`, license CC BY 4.0
- [ ] Paste DOI + URL back into repo files
